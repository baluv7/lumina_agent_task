HOW TO RUN: LangGraph Agent Assistant

1. Create a Python virtual environment (optional but recommended):

   For Windows:
       python -m venv env
       env\Scripts\activate

   For macOS/Linux:
       python3 -m venv env
       source env/bin/activate

2. Install the required Python libraries:

       pip install langgraph langchain langchain-community

3. Install and start Ollama:

   a) Download Ollama from:
       https://ollama.com/download

   b) Pull the Mistral model:
       ollama pull mistral

   c) Run Ollama (if not already running):
       ollama run mistral

4. Save your Python code as:
       agent_graph.py

5. Run the application:

       python agent_graph.py

6. Follow the on-screen menu to interact with the assistant.
